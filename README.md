# SimpleGame_iOS
Very Simple Game with Obj-C <br>
옛날에 만들었던 앱이라 자잘한 화면 사이즈가 안 맞는다 <br>
참고로 이미지는 제가 만든게 아니라 상업적 이용이 불가능 합니다.<br>
사실 문법을 익히기 위한 앱이라 효율도 안좋은 편이라 문법 공부용 코드 입니다. <br>
